# Graduation Project – System Overview

## Project Goal
Developed a centralized platform for managing **room reservations** across the **University of Sousse** and its **17 establishments**.

## Technologies Used
- **Frontend:** Angular 18
- **Backend:** Spring Boot
- **Database:** MongoDB
- **Mobile:** Android (if applicable)

## My Role
- Frontend development (Angular components and UI design)
- Authentication and user interface implementation
- Collaborating with backend developers

## Key Features
- User authentication and profile management
- Viewing and booking available rooms in 17 establishments
- Admin panel for managing reservations and requests
- Secure handling of data and user information

> ⚠️ Note: The source code is not publicly available due to security and confidentiality reasons.

- Applied full-stack development principles in a real project

- Gained experience with Angular, Spring Boot, and MongoDB
